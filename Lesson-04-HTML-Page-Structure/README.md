# Lesson 04 - HTML Page Structure (Your First Web Page)

In this lesson, we will create our **first HTML page** and learn the basic structure of any web page.

### What you will learn:
- Create a new project and open the folder in VS Code  
- Create an `index.html` file  
- Use the shortcut to generate the basic HTML structure  
- Understand page elements:
  - `<!DOCTYPE html>`  
  - `<html>`  
  - `<head>`  
  - `<body>`  
- Difference between information visible to users and internal metadata  
- Write your first heading and paragraph  
- Run the page using **Live Server**  

> This lesson is very important because it forms the foundation for everything else in the course.
