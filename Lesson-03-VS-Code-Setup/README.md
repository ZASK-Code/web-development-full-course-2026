# Lesson 03 - VS Code Setup for Beginners + 3 Must-Have Extensions

In this lesson, we will set up **VS Code**, the best code editor for web development, and install 3 essential extensions that will help us write code more efficiently:

### Steps:

1. **Install VS Code**  
   - Direct download link is available in the YouTube video description.  

2. **Install Extensions**:  
   - **Live Server:** Run your web page directly in the browser.  
   - **Prettier:** Automatically formats and organizes your code.  
   - **Auto Rename Tag:** Automatically renames opening/closing HTML tags when you edit one.  

> These are the most important extensions for HTML.  
> We will add more extensions when we start CSS and JavaScript.
