# Lesson 02 - How the Web Works (Internet vs Web Explained)

In this lesson, we will learn how websites work behind the scenes.

When you type a website name in Google and press Enter, how does it open in seconds? 🤔  
To understand, we need to clarify some concepts:

- **WWW (World Wide Web):** A global system that links documents via **hyperlinks**.  
- **Web server:** A large system where websites are stored.  
- **Web browser:** A program like Chrome or Edge used to display web pages from the server.  
- **Communication:** Browser and server communicate using **HTTP (Hypertext Transfer Protocol)**.  

**Difference between the Internet and the Web:**  
- **Internet:** A huge global network of connected devices.  
- **Web:** A service on the Internet that lets us access information via web pages.  

Example: When you type `google.com` in your browser:  
1. Your request travels through the Internet to Google’s server.  
2. The server sends back the pages (HTML, CSS, JS).  
3. The browser assembles the pages to display the website you see.  

In this course, we will learn **how to build these pages ourselves**.
