# Lesson 06 - Anchor Tags (Links)

In this lesson, we will learn **Anchor Tags** to create links inside and outside your website.

### What you will learn:
- **External links**: Link to other websites using `<a href="URL">text</a>`  
- **Internal links**: Link to pages within your website for navigation  
- Creating multiple pages (e.g., index.html, about.html)  
- How to organize your website with a **navigation bar**
