# Lesson 05 - HTML Headings & Paragraphs

In this lesson, we will learn **HTML Headings and Paragraphs** to build clean and organized web pages.

### What you will learn:
- The 6 levels of headings: `<h1>` to `<h6>`  
  - `<h1>`: Main heading / most important  
  - `<h2>` to `<h6>`: Subheadings for sections and subsections  
- How headings indicate content hierarchy (not just size)  
- The importance of headings for SEO and user readability  
- Adding paragraphs using `<p>` tags  
- Optional: using `<hr>` to separate sections, `<br>` for line breaks
